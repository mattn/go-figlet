# flf-3d

Generates 3D shadow FIGlet fonts from existing FLF files. Each filled pixel becomes `‚Ėą` with a `‚ĖĎ` shadow at the lower-left.

## Usage

```
flf-3d input.flf > output.flf
```

## Example

```
$ flf-3d efont-b12_b.flf > 3d-efont-b12.flf
$ go-figlet -font 3d-efont-b12.flf "figlet"
```
