# flf-shadow

Generates ANSI Shadow style FIGlet fonts from existing FLF files. Each filled pixel becomes `‚Ėą‚Ėą` with box-drawing border characters (`‚ēĒ`, `‚ēź`, `‚ēó`, `‚ēĎ`, `‚ēö`, `‚ēĚ`) at the edges.

## Usage

```
flf-shadow input.flf > output.flf
```

## Example

```
$ flf-shadow efont-b12_b.flf > shadow-efont-b12.flf
$ go-figlet -font shadow-efont-b12.flf "figlet"
```
